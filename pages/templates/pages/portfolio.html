Project 1

UniCare AI - 유학생 맞춤형 다국어 심리상담 및 캠퍼스 지원 AI 웹 서비스

팀명: Pioneer

지도교수: 전숙연, 장진수

연구사업: 캡스톤 디자인

개발배경

한국의 유학생 수가 증가하면서 언어 장벽, 학업, 사회적 통합 등의 어려움을 겪고 있으며, 기존 상담 시스템의 한계로 인해 AI 기반 맞춤형 상담 서비스의 필요성이 대두되었습니다.

개발목표

생성형 AI와 로컬 LLM을 활용하여 유학생을 위한 맞춤형 다국어 AI 상담 지원 웹 서비스를 개발하는 것을 목표로 합니다.

주요 기능
  • AI 상담 챗봇
  • 음성 대화 기능
  • VR 기능
  • 감정 표현 및 추적
기술 스택
LangChain LangGraph OpenAI API Web Development Database AI Integration
연계 교과목
데이터베이스 캡스톤디자인 인공지능의 이해
기대효과
  • • 대학 상담센터 업무 보조
  • • 유학생 정서 안정 및 학업 성취도 향상 지원
  • • 상담 인력 부담 감소
  • • 글로벌 교육 서비스 경쟁력 강화

기관: 전주비전대학교 (Jeonju Vision University)

Project 2

미얀마어 대학 홈페이지 개발 (Myanmar Language University Homepage)

프로젝트 개요

팀 프로젝트의 일환으로 미얀마어를 사용한 대학 홈페이지를 개발하였습니다. 이 프로젝트는 한국 대학에 대한 정보를 미얀마어 사용자들이 쉽게 접근할 수 있도록 하는 것을 목표로 하였습니다.

주요 기여
  • 사이트 구조 설계
  • 미얀마어 콘텐츠 구현
  • 사용자 친화적인 커뮤니케이션 디자인
  • 다국어 현지화 작업
주요 기능
  • 미얀마어 네비게이션 (အိမ်, အကြောင်း, ဌာနများ)
  • 홈페이지 (학사복 차림 학생들 이미지)
  • 학과 소개 페이지 (Child Welfare, Social Welfare Management 등)
  • 대학 정보 제공
기술 스택
Web Development Localization Multilingual Support UI/UX Design
학습한 내용
  • • 협업 능력 향상
  • • 웹 콘텐츠의 다국어 현지화 방법
  • • 사용자 중심의 웹사이트 설계
  • • 명확하고 효과적인 커뮤니케이션 디자인

프로젝트 유형: 팀 프로젝트

대상 사용자: 미얀마어 사용자
